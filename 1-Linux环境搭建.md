> 开源的大部分都是免费的。如何理解开源呢？

1. 为什么有人愿意无偿参与？

   最初是几十年前贡献开源代码的人自己不差钱；有些开源是一件改变世界的事情，比如Linux；第三点是和社会的志愿者是一类角色，世界上有很多顶级的工程师愿意参与。

2. 开源 VS 闭源

   对于开源来说，简单说优点就是集思广益，参与人数很多，有人写，有人用，有人看，有人发现问题，有人解决问题，慢慢地就会出现越来越好的代码。
   
   对于闭源来说，参与人数有限，在发现问题和解决问题的速度肯定没有开源快，效率一定没有开源高。
   
   我们从安全方面说，开源也就是所有的逻辑公开，源码之前了无秘密。

> Linux操作系统有各种各样的版本

1. 商业化发行版：Ubuntu，CentOS……

   可以看成 Linux 操作系统的品牌

2. 技术发行版：1.XXX或者2.XXX等等

![image-20240804144534987](1-Linux环境搭建.assets/image-20240804144534987.png)

修正次数达到某个数字会向次版本号进 1，次版本号达到某个数字会向主版本号进 1。

如果次版本号是偶数，说明该版本是稳定版本。如果是奇数，表示测试版本，测试版本发出了先用一用，然后观察一段时间，没有问题再升级到偶数，就像抢先体验。

## 1.2 Linux环境搭建（购买云服务器）

> 有三种方式

1. 双系统（不推荐）

   在 Windows 再装一个系统，成本很高，在装双系统的时候 Windows 容易出问题。

2. powershell（不推荐）

3. 虚拟机 + Ubuntu 镜像（比较推荐）

   需要下载虚拟机软件，推荐 VMware player 免费版 + CentOS 7/Ubuntu镜像文件，由于CentOS停止更新维护，采用Ubuntu即可

   将在后面详细说明。

4. 云服务器（最推荐）

   云服务器是真正的开发和生产环境，常见的有阿里云，腾讯云，华为云以及百度云

> 购买云服务器，以腾讯云为例
>

- 选择轻量应用服务器

  ![image-20240804144523072](1-Linux环境搭建.assets/image-20240804144523072.png)

- 进入到特惠

  ![image-20240804144514202](1-Linux环境搭建.assets/image-20240804144514202.png)

- 选择 2核2G 基本够用了，当然你也可以选择 2核4G 的

  ![image-20240804144501443](1-Linux环境搭建.assets/image-20240804144501443.png)

- 进去后选择常见的 CentOS 或者 Ubuntu，由于 CentOS 停止维护，所以建议选择Ubuntu 22.04/20.04/24.04 LTS 即可。

- 购买完成后，点击右上角 控制台

  ![image-20240804144451579](1-Linux环境搭建.assets/image-20240804144451579.png)

- 点击 轻量应用服务器

  ![image-20240804144441366](1-Linux环境搭建.assets/image-20240804144441366.png)

- 进去后一定要 重置密码，密码尽量复杂

  ![image-20240804144431387](1-Linux环境搭建.assets/image-20240804144431387.png)

- 重置好密码后，点击确认即可

  ![image-20240804144419169](1-Linux环境搭建.assets/image-20240804144419169.png)

> 购买完成后，怎么样登录云服务器？

1. 复制你的公网IP

   ![image-20240804144402738](1-Linux环境搭建.assets/image-20240804144402738.png)

2. 下载 Xshell，选择家庭学校免费版

   ![image-20240804144346669](1-Linux环境搭建.assets/image-20240804144346669.png)

3. 安装完成后，登录云服务器，形式为ssh root@云服务器公网IP，然后回车，点击接受并保存（如果是Ubuntu的话，root需要写成ubuntu）

   ![image-20240804144321454](1-Linux环境搭建.assets/image-20240804144321454.png)

4. 出现输入密码框，密码就是前面购买云服务器后重置的密码

   ![image-20240804144304832](1-Linux环境搭建.assets/image-20240804144304832.png)

## 1.3 Linux环境搭建（虚拟机 + Ubuntu镜像文件）

> 下载 VMware 虚拟机

由于 VMware 被 Broad 收购，所以 VMware 下载链接等被并入 Broad 网站。点击[Broadcom Inc. | Connecting Everything](https://www.broadcom.com/)，进入到以下界面

![image-20240808185641289](1-Linux环境搭建.assets/image-20240808185641289.png)

点击 Register 注册，注册成功后返回首页，点击 Go To Portal 登录，username 就是注册时使用的邮箱。

![image-20240808190040616](1-Linux环境搭建.assets/image-20240808190040616.png)

登录成功后进行以下操作

![image-20240808190503104](1-Linux环境搭建.assets/image-20240808190503104.png)

往下滑，找到 VMware Workstation Pro

![image-20240808190543685](1-Linux环境搭建.assets/image-20240808190543685.png)

![image-20240808190646733](1-Linux环境搭建.assets/image-20240808190646733.png)

注意：如果你的电脑是 Windows，这里就下载 Windows，不要以为我们需要的是 Linux 从而下载错误。然后选版本，一般最新即可。

![image-20240808191435995](1-Linux环境搭建.assets/image-20240808191435995.png)

这个下载非常慢，可以考虑用一些方法让下载加快。

下载完成后，傻瓜式安装即可。但是有两个地方要注意：

![image-20240808195658398](1-Linux环境搭建.assets/image-20240808195658398.png)

建议所有软件都安装到C盘之外，DEF盘都行。

![image-20240808195804409](1-Linux环境搭建.assets/image-20240808195804409.png)

将这两个取消勾选。

> 虚拟机下载完成后，需要下载Ubuntu的镜像文件

点击[清华大学开源软件镜像站](https://mirrors.tuna.tsinghua.edu.cn/)，进去后搜索 Ubuntu，找到发行版

![image-20240808200433237](1-Linux环境搭建.assets/image-20240808200433237.png)

![image-20240808201627713](1-Linux环境搭建.assets/image-20240808201627713.png)

安装这些稳定版本都是可以的，如果你之前没有使用过，那么采用最新版本 24.04 是最好的选择，它将支持到 2029 年，如果是 22.04 和 20.04，将会支持到 2027 年和 2025年。我是22.04，其他版本也是一样的操作。

![image-20240808201951680](1-Linux环境搭建.assets/image-20240808201951680.png)

点击它，即可下载。下载完成后，打开 VMware —> 创建新的虚拟机 —> 点击下一步

![image-20240808204037250](1-Linux环境搭建.assets/image-20240808204037250.png) 

这里就选择你镜像文件下载的位置。

![image-20240808204159543](1-Linux环境搭建.assets/image-20240808204159543.png)

全名和密码随便写，用户名只能是小写字母、数字、破折号。然后一路傻瓜式安装。

![image-20240808210013730](1-Linux环境搭建.assets/image-20240808210013730.png)

这里看情况可以改大，也可以不改。

![image-20240808204908104](1-Linux环境搭建.assets/image-20240808204908104.png)

到这里建议自定义硬件，每个处理器的内核数量修改为 2

![image-20240808205010408](1-Linux环境搭建.assets/image-20240808205010408.png)

![image-20240808210408110](1-Linux环境搭建.assets/image-20240808210408110.png)

网络适配器选择桥接模式。

![image-20240808210502483](1-Linux环境搭建.assets/image-20240808210502483.png)

这里的勾选取消掉，点击完成，如果出现以下界面，说明虚拟机创建完成。

![image-20240808210558098](1-Linux环境搭建.assets/image-20240808210558098.png)

![image-20240808210638260](1-Linux环境搭建.assets/image-20240808210638260.png)

点击期中任意一个进入虚拟机。稍微等一会儿即可。

到这里是对虚拟机的创建，创建后对Ubuntu的系统安装等又是另一件事情了。
